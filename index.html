Assignment 1 - Basic HTML

Assignment 1 asked us to create a webpage of any topic of our choice that utilized the correct doctype and structural code, properly formatted HTML code, one HTML comment with our name and the course, a title tag, a main heading (h1), two paragraphs, two images, one numeric or bulleted list, one table, and two external links.

Assignment 2 - Basic CSS

Assignment 2 asked us to create a webpage of any topic of our choice that included an index page, a title (h1), a navigation bar with four different links, a sub header (h2), two paragraphs, two pictures, and a table. This assignment also required us to utilize CSS to change the font of the page; style the page with an appealing color scheme; format the navigation links; make the navigation links change format when hovered over; add borders and padding around the images; add padding, borders, and background colors to the table; add a body background image; use a formatted div to make the page content more readable; and use elements, ids, and classes to style the page.

Assignment 3 - Page Layout

Assignment 3 asked us to create a webpage with a desired navbar and column layout using Flexbox, Media Query, and CSS.

Assignment 4 - Recreate CSS

Assignment 4 asked us to recreate a webpage for both large and small screens using CSS and Flexbox.

Assignment 5 - JavaScript Buttons, Functions, and More

Assignment 5 asked us to create a page with three columns and have each column run a different function using JavaScript arrow functions and constants.
